##🌸 Puķu veidi Latvijā

1. `Viengadīgās puķes`
2. `Divgadīgās puķes`
3. `Daudzgadīgās puķes`


### Viengadīgās puķes
Šīs puķes aug un zied tikai vienu sezonu. Tās ir viegli audzējamas un bieži tiek sētas tieši dobē.

### Divgadīgās puķes
Tās zied otrajā gadā pēc sēšanas, bieži tiek izmantotas dārza kompozīcijās.
